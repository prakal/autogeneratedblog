Welcome to Pranav's coding blog!

Hi, I am Pranav, a beginner web developer based in Canada using primarily JavaScript and Python. I am trying to get to a professional level of web development, and developing my portfolio to reflect that. I am a big fan of immersive learning, and believe that I absorb more information when I am breathing code/algorithms. I am a chemical engineer as well, and have an analytical approach to problems, and eager to hunt for solutions.

I have previously developed a web based platform for displaying, and recommending clothes on the Amazon store based on a person's measurements. For example, if you have a 32'' waist and 30'' long legs, only pants of those dimensions would be visible to you, rather than a generic S/M/L fit. I developed this project to sharpen my coding skills, and also to provide a service that I confirmed that people would use, by survey. Also, I saw potential for affiliate revenue. However, this did not work out, as I was using the Amazon Product Advertizing API (to be used only on blogs advertizing products) to populate my database, it was easy for them to reject API access before it got off the ground. Future reminder: Read through all terms and conditions before going out to build. I used Django(Python) and Javascript for this site.
